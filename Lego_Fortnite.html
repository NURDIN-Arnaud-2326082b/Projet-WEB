Lego dans Fortnite !

Le mode de jeu Lego qui est disponible dans Fortnite est un jeu d'artisanat de survie sortie le 7 décembre 2023, similiaire au jeu Minecraft pour plus d'exemples.

Gameplay:

LEGO® Fortnite est un jeu à bac à sable de survie dans lequel votre objectif est de créer votre village, de l'améliorer pour avoir plus de personnages Fortnite, et dont un qui sera votre guide principale a travers le monde.

Il existe 2 types de modes de jeux :

Mode Survie :

    Dans ce mode le jeux, le joueur est livré a lui même, il doit recolter des ressources pour pouvoir survivre face aux monstres, debloquer des nouveaux items (objet) comme des moyens pour combattre, farmer, construire etc, et developper son village. A ce jour, ce mode de jeux ne connais pas de fin precise mais selon une majorité de joueurs, une fin possible serait de finir de developper chaque village dans chaque biome (environnement) possinle en jeu.

Mode Creative :

    Dans ce mode de jeux, tout est deja debloquer, vous n'avez plus rien a farmer ou etc..., vous etes livré a votre inspirations, vous pouvew voler pour vous aider a construire.
